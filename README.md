Human-AI Workflow Cookbook

Practical workflows for using AI as a thinking partner, operator assistant, and execution tool.

This repository is a collection of repeatable human-AI workflows for research, job applications, project planning, content creation, interview prep, debugging, learning, and everyday execution.

The goal is not to collect random prompts.

The goal is to document useful processes that help people think clearly, make decisions, finish work, and use AI without losing human judgment.

Purpose

AI is most useful when it is used with structure.

A good workflow defines:

* what the human owns
* what the AI assists with
* what inputs are needed
* what output should be produced
* where judgment, review, and revision happen

This cookbook is built around that idea.

Core Principle

AI should not replace thinking.

AI should help organize thinking, challenge assumptions, draft options, reduce friction, and make execution easier.

The human remains responsible for judgment, context, ethics, and final decisions.

Standard Workflow Format

Each workflow follows this structure:

1. Goal
2. Best Used When
3. Inputs
4. Human Role
5. AI Role
6. Step-by-Step Process
7. Expected Output
8. Common Mistakes
9. Example Prompt
